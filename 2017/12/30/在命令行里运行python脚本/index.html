在命令行里运行python脚本

撰写于 2017-12-30 修改于 2017-12-30

< !–more–>
打开文本编辑器，notepad
语言设置成python
脚本第一行一定要写上 #! /usr/bin/env python
编辑好之后，将文件保存成 .py格式
cd 到运行文件所在目录
运行python 文件名.py
cmd命令行中输入 python
把脚本文件拖拽到当前光标位置，回车
报错：
File ““ , line 1
python hello.py
SyntaxError：invalid synta
原因：Python命令行中，运行Python代码
“python hello.py”是一个脚本命令，不是python代码
